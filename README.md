Projection Output — 使用说明 (Python 版)
目标：运行实时渲染脚本生成可直接投影的最终视觉效果（基于论文内容：远航 / 共生 / 脉动 三章）。

文件说明

luogang_projection.py：核心执行程序，基于 Python 环境驱动的实时视觉系统。

运行方式

环境准备： 确保系统已安装 Python 3.x。 安装必要的图形库依赖：

Bash
pip install pygame
启动程序： 在终端或命令行工具中进入文件所在目录，运行：

Bash
python luogang_projection.py
显示模式：

程序启动后将默认以 全屏模式 运行，以匹配投影仪输出。

按 Esc 键可直接退出程序。

实时交互快捷键

数字键 1：远航（视觉：冷蓝矢量线，象征探索与轨迹）。

数字键 2：共生（视觉：有机分支，象征生命力与连接）。

数字键 3：脉动（视觉：粒子场，随鼠标移动产生推力或引力）。

空格 (Space)：暂停/继续当前的实时演算。

鼠标移动：在“脉动”章节中模拟人群密度交互，干扰粒子流向。

投影配置建议

硬件分辨率： 建议将投影仪分辨率设置为系统主屏幕分辨率（如 1920x1080），脚本会自动捕捉当前屏幕尺寸。

性能保障： 由于是实时物理演算，建议关闭电脑的其他高负载程序，确保帧率稳定在 60fps。

备份预案： 若需在无 Python 环境的设备上播放，建议使用录屏软件将脚本运行画面录制为 .mp4 视频。
